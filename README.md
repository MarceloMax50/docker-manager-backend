# docker-manager-backend

Construir um sistema de gerenciamento e monitoramento de serviços baseado no Docker (API).

Atividade em grupo!

Documentação da API do Docker: https://docs.docker.com/engine/api/v1.41/#operation/ContainerList

O sistema deve seguir a arquitetura de referência abaixo:

![image](https://user-images.githubusercontent.com/49374397/141389930-ee0c5912-6492-4642-ba35-763d15ab5d9f.png)


A entrega mínima deve ser composta dos seguintes recursos:

![image](https://user-images.githubusercontent.com/49374397/141389957-732b6a8d-c098-4d65-9854-2615463cc69e.png)


O entregável deve apresentar uma interface de gerencimento (a escolha da linguagem de programação e o uso de frameworks é livre) e uma interface de monitoramento (grafana).
